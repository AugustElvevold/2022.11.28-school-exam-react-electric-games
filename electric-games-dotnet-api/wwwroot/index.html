    Response

    }]

    /api/Games

    Parameters

    Requested body:
    application/json
    {
    "id": "string",
    "title": "string",
    "platform": "string",
    "release_year": "string",
    "image": "string"
    }

    Response

    200

    Description

    Creates a new game in the database


  • Get

    /api/Games/{id}

    Parameters

    Path: id string

    Response

    {
    "id": "string",
    "title": "string",
    "platform": "string",
    "release_year": "string",
    "image": "string"
    }

    Description

    Returns a game if a matching id is found in the database


  • Put

    /api/Games/{id}

    Parameters

    Path: id string
    Requested body:
    {
    "id": "string",
    "title": "string",
    "platform": "string",
    "release_year": "string",
    "image": "string"
    }

    Response

    200

    Description

    Updates a game by id, if a matching id is found in the database


  • Delete

    /api/Games/{id}

    Parameters

    Path: id string

    Response

    200

    Description

    Deletes game by id, if a matching id is found in the database


  • Get

    /api/Games/search/{searchWord}

    Parameters

    Path: searchWord string

    Response

    [{
    "id": "string",
    "title": "string",
    "platform": "string",
    "release_year": "string",
    "image": "string"
    }]

    Description

    Returns all games in the database that has text matching the search word


  • Post

    /api/Games/UploadImage

    Parameters

    Requested body:
    multipart/form-data
    file string($binary)

    Response

    200

    Description

    Uploads an image to wwwroot/images/games
